# 🧪 _raw — 날것의 일일 학습 메모

여기는 **매일 공부한 것을 빠르게 적는 초안 공간**이다.
다듬을 필요 없이 생각나는 대로 적는다. 오타·반말·조각글 다 괜찮다.

## 흐름

```
_raw/2026-07-02.md   (오늘 배운 걸 날것으로 적음)
        │
        ▼  정리 (직접 또는 Claude에게 요청)
concepts/*.md        (재사용 가능한 개념으로)
journal/phaseN.md    (단계별 학습 노트로)
```

`_raw/`는 **원자재**, `concepts/`·`journal/`은 **완제품**이다.
원자재는 지우지 않고 그대로 둔다 (나중에 다시 볼 수 있게).

## 사용법

1. 하루 학습을 시작하며 [`_daily-template.md`](./_daily-template.md)를 복사한다.
   - 파일명: `YYYY-MM-DD.md` (예: `2026-07-02.md`)
2. 그날 배운 것·막힌 것·의문점을 자유롭게 적는다.
3. 어느 정도 쌓이면 위키 본문(`concepts/`, `journal/`)으로 정리해 옮긴다.
   - 정리한 메모에는 상단에 `> 정리 완료 → [링크](../concepts/xxx.md)`를 남겨두면 추적이 쉽다.

## 반영(정리) 가이드

- **개념이 잡혔다** → `concepts/`에 새 문서. `_template.md` 사용.
- **그 phase의 진행/깨달음이다** → `journal/phaseN.md`에 이어 붙임.
- 옮긴 뒤 `docs/README.md`의 문서 목록에서 "(작성 예정)"을 실제 링크로 바꾼다.

> 💡 Claude에게 "오늘 _raw 메모 정리해서 위키에 반영해줘"라고 요청하면
> 이 규칙에 따라 concepts/journal로 옮기고 인덱스까지 갱신해준다.
